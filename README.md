## 项目运行
安装工具
```
npm i -g yo &&
npm i -g generator-react-webpack
```
clone项目源码
```
git clone https://github.com/Rectmoon/gallery.git
```
安装依赖
```
cd gallery &&
npm install
```
运行
```
npm start
```
打开浏览器进入localhost:8080

#### 运行环境
node 6+ npm 4+
 
 
### 技术栈
- webpack：构建工具
- es6: 使用es6语法
- stylus: css预处理
- react+jsx
